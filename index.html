  • 컨테이너 기반 인프라 구축 — Docker로 Ollama와 OpenClaw를 통합 배포하고, 외부 통신 없이 독립적으로 작동하는 보안 에이전트 환경 구축
  • 모델 성능 벤치마킹 — 로컬 모델(Llama 3 등)과 외부 상용 모델(Claude 3.5 등)의 추론 정확도 및 응답 속도를 다각도로 비교 테스트
  • 리소스 효율성 분석 — 고사양 장비 상시 가동에 따른 전력 소모 및 하드웨어 유휴 리스크를 TCO 관점에서 분석
R Result

보안 이슈가 없는 완전 로컬 기반 에이전트 작동 확인. 다만, 장비 성능에 따른 속도 편차와 고사양 서버 유휴 상태 유지에 따른 비용 효율성 한계 도출.

K Takeaway
  • 엣지 컴퓨팅 접근 방식 — 단일 고사양 서버 의존 방식 대신, 모바일·저전력 기기 기반 경량 모델(SLM)을 활용한 분산 처리 아키텍처의 필요성 인식
  • 비용 최적화 전략 — 네트워크 기반 기기 간 리소스 공유 및 엣지 컴퓨팅 접목 시, 보안성을 유지하면서 인프라 비용을 최소화하는 효율적인 AI 시스템 설계가 가능할 것으로 판단
